# Portfolio - Facundo Romano

Sitio web personal para mostrar proyectos, habilidades y experiencia como desarrollador de software.

## Tecnologías

- HTML5 / CSS3
- JavaScript (ES6+)
- Bootstrap 5.3.2
- Bootstrap Icons
- Font Awesome 6.4
- Google Fonts (Inter, Space Grotesk, Manrope)
- EmailJS

## Estructura del Proyecto

```
portfolio/
├── index.html           # Página principal
├── styles/
│   └── styles.css       # Estilos CSS
├── scripts/
│   ├── script.js        # Funcionalidad principal
│   ├── cursor-effect.js # Efecto de cursor
│   └── validations.js   # Validaciones de formulario
└── images/              # Recursos gráficos
```

## Sitio Deployado

🔗 [facuroman0.netlify.app](https://facuroman0.netlify.app/)

## Funcionalidades

- Tema claro/oscuro con persistencia
- Diseño responsive
- Animaciones de entrada
- Navegación suave entre secciones
- Formulario de contacto funcional
- Logo SVG personalizado
